# Basic product walkthroughs
**Epic ID:** GF-59 | **Phase:** Phase II: Expanded Pilot | **Priority:** P2 — Medium | **Swim Lane:** Collaboration & Admin | **Owner:** Remy | **Build Status:** NEW

**Description:** Contextual tooltips and guided tours for key product features that appear when users encounter them for the first time. Covers map controls, list filtering, pipeline management, contact reveals, saved lists, and credit usage. Dismissable and non-blocking — power users can skip, but new users get gentle guidance. Reduces support burden and helps users discover features they might otherwise miss.

**Success Criteria:** Walkthroughs cover all major features (map, list, pipeline, contacts, lists, credits). Users can dismiss permanently. Feature discovery rates increase for walked-through features. Support ticket volume for 'how do I...' questions decreases.
