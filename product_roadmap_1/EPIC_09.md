# Segment-specific filter defaults
**Epic ID:** GF-9 | **Phase:** Phase II: Expanded Pilot | **Priority:** P2 — Medium | **Swim Lane:** Prospecting & Search | **Owner:** Remy | **Build Status:** NEW

**Description:** When a user or org selects their primary service type (landscaping, janitorial, HVAC, etc.), the default filter configuration adjusts to surface the most relevant properties. A landscaping company cares about lot size and permeable area; a janitorial company cares about building sqft and tenant count. This reduces setup friction and helps users find high-value properties faster without manually configuring every filter.

**Success Criteria:** Default filters differ meaningfully by segment. Users in landscaping see lot-size-heavy defaults; janitorial users see sqft-heavy defaults. Time to first useful search result decreases vs. generic defaults.
